Dependencies (local devnet)

Java 17+ and Hyperledger Besu are needed ONLY to run the optional local devnet. The wallet and UI work without them. Nothing is installed automatically — copy the exact fix command shown for anything missing. See ACAP_WINDOWS_DEPENDENCY_INSTALL_GUIDE.md for full steps.

Local private-devnet lab

QBFT proof-of-authority devnet (chain id 469469), RPC bound to 127.0.0.1 only. No public mainnet, no rewards, no public RPC. Use the START/STOP/CHECK_LOCAL_DEVNET scripts in this folder — they refuse to run on a server and reject any non-loopback RPC.

Bundled validator keys are NON-PRODUCTION LAB keys — not user wallet keys, not the production chain's keys, no real funds. Never reuse them on any public network.

Recommended install: download ACAP_INSTALL.bat from the portal and double-click it. It downloads ACAP + Java + Besu, verifies SHA-256, installs under %LOCALAPPDATA%\ACAP-Desktop-Node (no admin, no PATH edit), and starts the node.

Update check: open the portal node page for the latest package version & SHA-256. This shell does not auto-update.
