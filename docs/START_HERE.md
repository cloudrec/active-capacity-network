# Start here — Active Capacity Network (ACAP)

> **Private preview / prototype.** No public mainnet, no token sale, no custody,
> no payments, no bridge, no rewards. All chain-style data is demonstration data
> and clearly labeled. Nothing here moves real value.

## What this repository is

The **public preview package** for the Active Capacity Network (ACAP):

- **Concept + prototype** of a capacity protocol using **Proof of Active Capacity
  (PoAC)**. The token unit is **ACAP**.
- **Desktop node source** you can read, build, and run locally.
- **Windows bootstrap installer** (per-user, no admin).
- **Local private devnet** (loopback only, lab keys).
- **Tokenomics preview** (everything disabled, fixed hard cap).

## What is NOT live

Real bridge, public mainnet, custody, payments, guaranteed rewards/yield, token
trading, and a public sale are **not implemented** and must not be treated as
live. See [DISCLAIMERS.md](DISCLAIMERS.md).
