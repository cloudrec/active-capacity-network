## Tokenomics are preview

ACAP tokenomics are **draft / preview** and may change before any launch. Tiered
reservation pricing is a **capacity-pricing mechanism**, not a profit promise or
investment return. The canonical values live in
[`../tokenomics/acap_tokenomics.json`](../tokenomics/acap_tokenomics.json) and are

## Not financial advice

Nothing in this repository is investment, financial, legal, or tax advice, an offer
to sell, or a solicitation to buy any token or security. ACAP is presented as a
utility / capacity token **design**.

## Reservation model

Active Capacity Reservation is **not** a fixed-term lease: ACAP remains the
holder's; nothing expires or is confiscated after a term. Active-capacity *weight*
depends on movement, node use, and participation. Reservation is currently
disabled.

## Keys and safety

This repository contains no real private keys, mnemonics, credentials, `.env`
files, or database dumps. The devnet ships a **placeholder** lab key; users
generate their own. The node and installer never print or store private keys.
